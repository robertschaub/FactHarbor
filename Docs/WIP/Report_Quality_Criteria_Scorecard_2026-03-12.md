## 1. Document Role

This file keeps the reusable report-quality rubric that still helps compare families or review future reports. It no longer serves as the live benchmark scoreboard for old March runs.

---

## 2. Reusable General Criteria

| # | Criterion | Why it still matters |
|---|-----------|----------------------|
| **G1** | Verdict stability across repeated runs | Re-runs should not drift arbitrarily on the same claim |
| **G2** | Boundary count and boundary quality | Boundaries should add analytical value, not decorative complexity |
| **G3** | Claim decomposition accuracy | Distinct events or dimensions should not collapse into one misleading unit |
| **G4** | Evidence relevance | Foreign or off-topic evidence must not contaminate the verdict |
| **G5** | Confidence plausibility | Confidence should reflect evidence quality and uncertainty honestly |
| **G6** | Political contestation immunity | Unsupported rhetoric must not adjust verdicts |
