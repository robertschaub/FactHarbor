**FactHarbor: First Pre-Release Published**

I'm happy to announce the first pre-release of FactHarbor — an open-source platform for automated claim verification.

Submit any claim. The system decomposes it into atomic assertions, retrieves evidence from multiple sources, groups findings into analytical perspectives, and generates verdicts through adversarial LLM debate. Every conclusion is traceable to specific evidence and source excerpts.

FactHarbor is being developed as a Swiss nonprofit (Verein). The full source code is available on GitHub.

Try it: [https://app.factharbor.ch](https://app.factharbor.ch/)

Feedback, collaboration ideas, and referrals to researchers in NLP, computational social science, or fact-checking are welcome.
