# Pre-Release UI Texts — FactHarbor Alpha Launch

**Date:** 2026-03-02
**Status:** Draft — ready for review
**Purpose:** All user-facing text for UI components at pre-release launch. No code — text content only.

---

## 1. Disclaimer Banner

Persistent top banner, every page. Must fit a single line on desktop (~120 characters target).

**Primary (recommended):**
> FactHarbor Alpha — AI-generated analysis may contain errors. Not a substitute for professional fact-checking. No warranty is provided.

**Alternate A (slightly warmer):**
> Early Access: Results are AI-generated and may be imperfect. Use critical judgment — this is not professional fact-checking advice.

**Alternate B (most compact):**
> Alpha: AI-generated results may contain errors. Not professional advice. No warranty provided.

---

## 2. Result Disclaimer

Inline, above analysis results on the job detail page. One sentence.

> This analysis was generated by AI using publicly available sources and may contain errors or omissions.

---

## 3. Methodology Note

Below the analyze input area. 3–4 sentences.

> When you submit a claim, question, or article, FactHarbor extracts the core verifiable assertions, researches them using live web sources, and subjects the evidence to a multi-step debate between an advocate, a challenger, and a reconciler before producing a verdict. Each conclusion is scored on a 7-point scale from TRUE to FALSE with an associated confidence level. Analysis typically takes 2–5 minutes depending on complexity. [Learn more at FactHarbor.ch](https://factharbor.ch)

## Notes for Implementation

- The disclaimer banner is NOT dismissible during pre-release phase (always visible). Dismissibility can be added post-launch.
- "Alpha" badge should appear adjacent to the product name/logo in the nav, not in the page body.
- The methodology note link `[Learn more at FactHarbor.ch](https://factharbor.ch)` should open in a new tab.
- All texts above use sentence case, not Title Case, for a professional tone consistent with technical accuracy.
